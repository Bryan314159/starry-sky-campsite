# 画风参考文档：罗小黑战记风格分析

## 一、为什么以《罗小黑战记》为画风参考

《罗小黑战记》是近十年来中国 2D 动画的标志性作品之一，由 MTJJ（木头）导演，经历了 TV 剧集（2011-2021）、大电影（2019）和续作电影（2025）的积累。其美术体系已经形成了一套高度自洽、可被系统分析的视觉语言。

本项目——星空营地——追求的核心画面感受（温暖、治愈、手绘感、质朴而不粗糙、沉浸式自然场景）与《罗小黑战记》的美学方向高度吻合。因此将其作为画风参考基准。

---

## 二、核心美学理念

> **"以柔克刚"的视觉哲学** —— 柔软线条、温润色彩、大量留白，这并非技术限制，而是自觉的审美选择。通过 2D 手绘的"绘画感"传递含蓄、从容与诗意。

> **"萌系外衣包裹严肃内核"** —— 画面可爱治愈，但不幼稚。角色和环境的设计服务于叙事和情绪，而非单纯的装饰。

---

## 三、造型风格

### 3.1 线条

| 特征 | 说明 |
|------|------|
| 线条类型 | 圆钝的粗线条，简练凝实 |
| 线条质感 | 手绘感、略有粗细变化，不是机械的矢量线条 |
| 轮廓处理 | 外轮廓清晰，内部细节靠色块而非线条区分 |
| 小黑耳朵 | 外轮廓线条做离散化处理，增强毛绒感 |

**对本项目的指导：** 3D 模型应使用 toon shader（卡通渲染）来模拟这种描边感。物体的边缘清晰，内部无复杂纹理堆叠。场景物体的轮廓应该"读得出来"，像画在纸上的线稿。

### 3.2 无影技法（无影画风）

这是罗小黑战记最核心的视觉特征之一：

- **人物不使用阴影** —— 大面积平涂色块填充，用单纯的颜色表现角色
- 不是"偷工减料"，而是 **"无影胜有影"** —— 对画师的要求反而更高，需要靠动作衔接和空间架构来体现立体感
- 代表同样技法的作品：今敏《千年女优》、新房昭之部分作品

**对本项目的指导：** 3D 场景中使用 `MeshToonMaterial`（卡通材质），它天然就是 2-3 阶色阶的平涂效果，不产生光滑渐变阴影。这是从"插画感"到 3D 场景最直接的翻译。

### 3.3 角色 vs 背景的对比关系

| 元素 | 处理方式 |
|------|---------|
| 人物/前景物体 | 简洁、圆润、平涂、少细节 |
| 背景/自然环境 | 精细、层次丰富、有手绘笔触 |

**"化繁为简"的设计哲学：** 角色简化，视觉焦点从角色放宽到整个场景。人物"简练"但不影响信息传达，甚至隐秘地塑造了对奇幻世界的认识。

**对本项目的指导：** 路牌柱、指示牌、星星这些"前景主体"保持简洁的造型；天空、草地、小路这些"环境背景"可以有更丰富的纹理层次。小猫角色尤其应该追求简洁可爱的造型。

## 四、色彩体系

### 4.1 总体方向

- **低饱和度** —— 不刺眼、不荧光、不"塑料感"
- **中国传统色彩基调** —— 蟹壳青、缟色、黛蓝、胭脂红、赭石、藤黄
- **温润克制** —— 像水彩画，不是油漆桶灌出来的纯色

### 4.2 场景色彩对照

| 场景类型 | 主色调 | 氛围 |
|---------|--------|------|
| 森林/自然 | 纯度较高的绿，层叠的叶片，从嫩绿到墨绿 | 心旷神怡，生命感 |
| 传统建筑 | 青砖灰瓦，低饱和暖棕，木色 | 安静、质朴、东方韵味 |
| 海洋/天空 | 黛蓝，天青，白云 | 深邃、宽广、诗意 |
| 城市夜景 | 明亮灯光，冷灰调建筑 | 繁华与疏离的对比 |
| 篝火/温暖场景 | 暖橙红，柔和的火光 | 治愈、安全、温暖 |

### 4.3 本项目色彩映射

| 场景 | 主色调 | 辅助色 | 参考罗小黑场景 |
|------|--------|--------|-------------|
| 场景一（营地·路牌） | 暖草绿、木棕、奶油白 | 浅橙、淡金、柔蓝（傍晚天空） | 森林场景 + 篝火夜谈的暖色调 |
| 场景二（星空·书签） | 深蓝紫、藏蓝、黛蓝 | 暖金（星光）、奶油白（文字） | 海洋夜空 + "醉里不知天在水"的星空意境 |
| 过场动画 | 暖色→冷色的渐变过渡 | — | 日夜交替氛围 |

### 4.4 色彩的功能性

动画中的色彩不仅是装饰，还有叙事功能：

- **烘托气氛** — 暖色 = 安全/温暖，冷色 = 宁静/辽阔
- **衬托角色** — 角色色块简洁，靠环境色衬托
- **展示时间** — 傍晚的天空渐变是时间的刻度
- **展现文化背景** — 低饱和中国传统色 = 东方美学质感

## 五、场景设计

### 5.1 森林场景（重点参考）

罗小黑电影开篇的森林被多位评论者称为"影片最大的视觉亮点"：

- 大片的森林，层叠的绿叶，树木细节刻画到位
- 阳光从树叶缝隙洒落，精灵在丛林中跳跃
- 纯度很高的绿色调，令人心旷神怡
- 背景画师耗时 3 年完成一千多张背景，场景在国内实地取材

**对本项目的指导：** 营地场景的草地和周围树木应该追求这种"层次丰富的绿"——不是一片纯色，而是有深浅、有斑驳、有手绘笔触的绿色地毯。

### 5.2 星空夜景

- 星星倒映在大海上的画面被形容为"醉里不知天在水，满船清梦压星河"
- 月色、星光的处理温柔不刺眼
- 夜景不追求"黑"，而是追求"深"——深蓝、深紫的层次渐变

**对本项目的指导：** 星空场景不应该是"黑底 + 白点"。天空应该有深蓝到深紫的柔和渐变，星星的暖金色光芒与深色天空形成冷暖对比。

### 5.3 建筑与物件

- 青砖、灰瓦、柱础、雕梁、马头墙——南方水乡建筑特色
- 木材质感：保留原木的纹理和色差，不追求"光滑完美"
- 手绘街招、书法字体——字也是画面的一部分

**对本项目的指导：** 路牌柱应该是"质朴原木"——看得见木纹，颜色有深浅变化，不是均匀的棕色塑料管。指示牌上的文字应该像手写在木板上，不是打印体。

## 六、氛围与情绪

### 6.1 治愈系（ healing-style ）

《罗小黑战记》被媒体和观众誉为"国漫治愈天花板"。治愈感的来源：

- 不是"煽情"，而是"安静的陪伴"
- 低饱和色彩 → 视觉不疲劳
- 自然环境的大量描绘 → 逃离都市的放松感
- 角色的萌感 → 柔软的情感连接
- 留白与节奏 → 给观众呼吸的空间

### 6.2 万物有灵的诗意

- 山林鸟鸣虫啼、茶馆檐角雨滴垂落
- 每一帧都藏着自然灵动的细节
- "做减法的智慧"——不是堆满画面，而是留出让人沉浸的空间

**对本项目的指导：** 场景中添加微小的环境动画——草丛微微晃动、偶尔飘过萤火虫光点、星星缓慢明暗呼吸。这些"不经意的小动作"正是治愈感的来源。

## 七、对本项目的具体设计指导

### 7.1 路牌场景（营地）

```
造型：圆柱木头路牌 + 钉着的木板指示牌
线条：toon 描边，木头边缘清晰可见
色彩：暖棕色木柱（有纹理深浅变化），木板偏米白/奶油色
      草地 = 层次绿（深绿→浅绿，斑驳笔触）
      天空 = 傍晚暖渐变（浅橙→淡金→柔蓝）
质感：木头有年轮纹理，草地有不规则斑点，天空有水彩晕染感
氛围：温暖、傍晚、出发前的期待
```

### 7.2 星空场景

```
造型：球体星星（大小不一），圆润饱满
线条：星星有微微的光晕（不是锐利的光点）
色彩：天空深蓝→深紫渐变，星星暖金/暖白
      地面保持绿色但偏暗（被夜色笼罩）
质感：星星有轻微的发光呼吸感
氛围：宁静、深邃、仰望的敬畏感
```

### 7.3 小猫角色

```
造型：圆润的几何体拼接（球形头 + 椭圆身体 + 小三角耳朵）
      或简单的剪影轮廓
线条：粗圆描边
色彩：小黑猫 = 深灰/黑色，可以带一点点暖色反光
      眼睛 = 大而明亮，是画面的情感焦点
质感：平涂为主，耳朵边缘可以有毛绒感的暗示
风格：萌但不幼稚，像是从罗小黑世界里走出来的角色
```

### 7.4 文字与 UI

```
指示牌文字：手写体/手绘感，仿佛用粉笔或白漆写在木板上
搜索框：半透明，融入场景色调，不突兀
提示文字：暖白/奶油色，柔和圆角
字体方向：无衬线但有手工感（不是机械的黑体/宋体）
```

## 八、画风执行清单

下面是将罗小黑风格转化到本项目时需要把握的关键点：

| # | 要点 | 做 | 不做 |
|---|------|-----|------|
| 1 | 色彩饱和度 | 低饱和、温润、中国传统色 | 荧光色、高饱和、纯原色 |
| 2 | 阴影处理 | 平涂色块、toon 2-3 阶色阶 | 光滑渐变、写实阴影 |
| 3 | 线条 | 粗圆描边、手绘感 | 锐利细线、完美矢量线 |
| 4 | 前景 vs 背景 | 前景简洁、背景有层次 | 前景堆砌细节 |
| 5 | 材质 | 木纹、纸张纹理、水彩渐变 | 金属反光、光滑塑料 |
| 6 | 氛围 | 温暖、治愈、有留白 | 拥挤、信息轰炸 |
| 7 | 萌感 | 圆润、小巧、可爱 | 写实、冷峻 |
| 8 | 文字 | 手工感、融入场景 | 机械字体、突兀 UI |
