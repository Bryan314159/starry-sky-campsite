# 收藏夹可视化 — 项目梗概

## 一句话概述

将浏览器书签从一个枯燥的列表，变成一段从营地走向星空的沉浸式视觉旅程。

## 形态

**浏览器新标签页插件/扩展**，安装后替代浏览器默认的新标签页，用户每次打开新标签页即进入这个场景。

## 数据来源

浏览器原生书签（Bookmarks），层级关系：

```
书签栏
├── 分类文件夹A（一级）
│   ├── 网站1（二级）
│   ├── 网站2（二级）
│   └── ...
├── 分类文件夹B（一级）
│   ├── 网站3（二级）
│   └── ...
└── ...
```

- **一级**：书签分类文件夹 → 映射到场景一的**路牌指示牌**
- **二级**：文件夹内的具体书签（网站名 + 网址）→ 映射到场景二的**星星**

## 场景设计

### 场景一：营地 · 路牌

**画面构成：**

- **地面**：草地/草坪，覆盖画面底部
- **核心元素**：画面中央偏左，一根木质路牌柱，柱上钉着多块指示牌，各指向不同方向（经典造型）
- **路径**：路牌前方，一条小路向画面深处延伸，具有纵深感
- **氛围**：温暖的营地感，白天或傍晚色调

**内容映射：**

- 每块指示牌显示一个书签分类（一级文件夹）的名称
- 指示牌的指向方向各不相同，营造"多条路可选"的感觉

**交互：**

- 点击某块指示牌 → 触发进入场景二的过场动画

---

### 过场动画：沿路前行

- 一只小猫角色出现在画面中
- 小猫从前景出发，沿着小路背对观众向前走
- 逐渐走向画面深处（透视缩小）
- 走到小路尽头 → 动画结束，切换至场景二

---

### 场景二：星空 · 书签

**画面构成：**

- **视角**：小猫站在小路尽头的视角，抬头望向天空
- **天空**：夜空，布满星星
- **氛围**：宁静、开阔的星空夜景

**内容映射：**

- 每颗星星 = 所点分类下的一个具体书签（网站名 + 网址）
- 星星大小不一，营造自然的星空感
- 当前阶段：大小随机分布
- 后续阶段：按书签使用频率动态调整大小

**交互：**

- **点击星星** → 浏览器打开对应网址（新标签页或当前页跳转）
- **Hover（可选）** → 显示网站名称，可能伴随星星微亮/放大效果

---

### 返回机制

- 在场景二中，点击小猫**身后的小路** → 触发反向动画，返回场景一（营地路牌页）
- 回到场景一后，用户可再次选择其他分类

## 视觉风格关键词

- 手绘/插画风 或 低多边形（Low Poly）3D 风
- 温暖的营地氛围 vs 宁静的星空氛围
- 叙事感、沉浸感
- 非传统UI，强调场景化交互

## 技术约束

- 必须作为浏览器扩展（Chrome/Edge/Firefox）运行
- 需要读取浏览器书签 API
- 需要渲染 3D 场景（Three.js 或 CSS 3D）
- 新标签页场景，要求加载速度快、资源轻量
