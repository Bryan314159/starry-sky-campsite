# 场景画面设计文档

## 一、构图原则

### 1.1 天空/地面比例

采用三分法构图，**地面为主体**，地平线置于画面上方约 1/3 处。

| 区域 | 占比 | 说明 |
|------|:---:|------|
| 天空 | ~35% | 暖色调傍晚渐变，烘托氛围不抢戏 |
| 地面 | ~65% | 营地叙事主力，容纳路牌、小路、草地 |

**依据：** John Thomas Smith (1797) "Rule of Thirds"——当地面是画面主要内容时，地平线应置于上方 1/3，地面占 2/3。50/50 等分会导致观众视线"悬空"，分散注意力。

### 1.2 视觉层级

```
第1层（焦点）：路牌柱 + 指示牌
    ↓   位置：画面右侧 1/3 黄金分割点
    ↓   特征：最大、最亮、细节最多
    ↓
第2层（引导线）：小路
    ↓   位置：画面底部中央 → 向纵深延伸
    ↓   特征：一点透视，逐渐变窄
    ↓
第3层（氛围）：天空 + 草地
        特征：烘托情绪，不过度抢夺注意力
```

### 1.3 构图模式：金字塔 + 引导线

```
        ┌───────────────┐
        │    天空       │
        │  ───────     │ ← 地平线（上方 1/3）
        │         ╲    │
        │    草地   ╲  │
        │       ┌─┐  ╲│
        │  小路 │路│   │ ← 路牌在右侧黄金位
        │  ════╡牌│   │
        │  ╱    │柱│   │
        │ ╱     └─┘   │
        │╱  前景      │
        └───────────────┘
```

## 二、相机设定

| 参数 | 值 | 说明 |
|------|-----|------|
| 位置 | (0, 1.5, 5.0) | 成人站立视线高度，距路牌约 5 米 |
| 注视点 | (0, 1.0, -2.0) | 微微向下看，视线落于路牌区域 |
| FOV | 60° | 自然视角，不拉伸不变形 |

**感受：** 用户"站在营地入口"，路牌在前方几米处清晰可见，小路从脚下延伸至远方。画面正面朝向用户，不偏不倚。

## 三、场景元素布局

### 3.1 元素位置表

| 元素 | 3D 坐标 | 画面位置 | 视觉层级 |
|------|---------|---------|:---:|
| 天空球 | 包裹全场，center(0,0,0), r=20 | 画面上方 35% | 3 |
| 草地 | y=0, 12×20 平面 | 画面下方 65% | 3 |
| 小路 | y=0.01, z: 3→-8, 宽 1.5 | 底部中央 → 深处 | 2 |
| **路牌柱** | **(2.5, 1.75, 0)** | **右侧 33% 黄金点** | **1** |
| 指示牌 | 柱上 y: 0.6+0.45×i | 纵向排列，各指不同方向 | 1 |

### 3.2 路牌柱 — 视觉焦点

路牌柱是场景一的绝对主角：

- **水平位置：** 画面右侧约 1/3 处（九宫格右竖线）
- **垂直范围：** 从地面延伸到画面上半部（柱高 3.5m）
- **姿态：** 矗立在草地上，原木质感，未经精细打磨
- **指示牌排列：** 从下到上纵向分布，每块指向不同角度（-0.6 + i×0.25 rad）

### 3.3 小路 — 视觉引导线

- **起点：** 画面底部中央偏下（用户脚下，z ≈ 3）
- **终点：** 向画面深处延伸消失（z ≈ -8）
- **形态：** 泥土小径，前景宽 → 远景窄（一点透视）
- **弯曲：** 轻微自然弯曲，不笔直（更像真实山路）

### 3.4 天空 — 氛围层

- 场景一：暖色调傍晚渐变（浅橙 → 淡金 → 柔蓝）
- 场景二：深蓝紫夜空渐变
- 使用 `MeshBasicMaterial`（不受场景光照影响，保持纯色氛围）

## 四、光照设计

### 场景一：营地·路牌

```js
ambientLight      intensity: 0.4                        // 基础环境光
directionalLight  position: [5, 8, 3], color: #ffe4b5  // 暖金傍晚阳光，斜照
hemisphereLight   sky: #87ceeb, ground: #4b7b3d          // 天空蓝 + 草地绿环境光
```

- 主光从右上方斜射，模拟傍晚阳光透过树冠的效果
- 暖金色方向光照亮路牌柱的正面（用户看到的一面）
- 环境光保持柔和，不产生强烈阴影

### 场景二：星空·书签

```js
ambientLight      intensity: 0.15   // 夜景极低环境光
// 星星自身发光（emissive），不依赖外部光源
```

## 六、空状态设计

无书签时，路牌柱上显示 3 块占位指示牌，颜色略淡于正常指示牌，显示引导文案：

1. "在这里添加你的第一个书签"
2. "在书签栏创建文件夹来整理你的收藏"
3. "然后回到这里，探索你的星空"

## 七、两场景画面参数对照

| 参数 | 场景一（营地） | 场景二（星空） |
|------|--------------|--------------|
| 相机位置 | (0, 1.5, 5.0) | (0, 0.2, 0) |
| 注视点 | (0, 1.0, -2.0) | 仰视天空 |
| FOV | 60° | 70° |
| 天空占比 | ~35% | ~75% |
| 地面占比 | ~65% | ~25% |
| 色调 | 暖色 | 冷色 |
| 光照 | 亮（傍晚阳光） | 暗（星光自发光） |
