# 场景一背景图 — 提供要求清单

**目的：** 给"将提供背景图的人"看的规格说明，确保图与 3D 浮层（路牌 / 篝火 / 萤火虫 / 小鸟 / 小路）能正确叠加。
**适用：** 任务 2.21（准备图资源）
**关联：**
- `doc/decisions/ADR-004-scene1-photo-plus-3d-overlay.md` — "静态图 + 3D 浮层"方案
- `spec.md` §一 SP1.1 / SP1.7 / SP1.8 / SP1.11 — 验收场景
- `doc/art-style.md §4.3` — 场景一色系基准
- `doc/scene-design.md` — 场景一构图

---

## 0. 速读 TL;DR

给我 **1 张**或 **3-5 张**：
> **暖色黄昏 / 湖边草地 / 地平线 1/3 位置 / 3D 路牌柱位置留空 / 3840×2160 横图 / 低饱和度治愈系手绘感 / 版权清晰**。

我会做 4K 放大（如需）→ 转 webp → 视觉锚点核对 → 放入 `public/scenes/campsite-bg.webp`。

## 1. 硬性规格（必须满足）

| # | 项 | 要求 | 原因 / 来源 |
|---|----|------|------------|
| H1 | **尺寸** | **3840 × 2160 px**（4K 横向 / 16:9）| 任务 2.22 强制（spec SP1.7）；覆盖 4K 显示器不糊 |
| H2 | **格式** | `.webp`（或 `.png` / `.jpg`，由我转 webp）| spec SP1.1 命名 `campsite-bg.webp` |
| H3 | **地平线位置** | 画面**自下而上约 1/3** 处（不要正中，也不要太靠上）| spec SP1.1 明确；正中是业余感 |
| H4 | **构图空区** | 地平线**正中央**保留一块相对"干净"的区域（容许 3D 路牌柱插入）| 任务 2.23 标定要求 |
| H5 | **画面分区** | 明确分出 **天空区**（上 2/3）+ **地面区**（下 1/3），二者边界柔和但可辨 | 3D 元素要按"天空 vs 地面"分层 |
| H6 | **版权** | 你的原创 / 你有商用授权 / 公有领域 | 任务 2.21 风险"图版权问题"由你确认（ADR-004 §风险）|
| H7 | **方向** | 横向（landscape），16:9 | 不要纵向、不要正方形 |

## 2. 色系 / 氛围（与 `art-style.md §4.3` 对齐）

| # | 项 | 要求 | 不接受 |
|---|----|------|--------|
| C1 | **整体色相** | 暖色系（傍晚 / 黄昏）| ❌ 正午、❌ 深夜、❌ 雪景、❌ 阴雨 |
| C2 | **饱和度** | 低饱和度（治愈系）| ❌ 荧光色、❌ 纯原色（红/蓝/绿冲）|
| C3 | **天空区色相** | 暖橘 → 淡紫 / 淡蓝的**柔和渐变**| ❌ 纯橘、❌ 纯蓝、❌ 死黑 |
| C4 | **地面区色相** | 草绿（偏暖、不要冷绿）+ 木棕点缀 | ❌ 冷绿、❌ 黄绿荧光 |
| C5 | **氛围** | "治愈 / 静谧 / 温暖"——参考《罗小黑战记》傍晚场景 | ❌ 阴森、❌ 喧闹、❌ 戏剧化 |
| C6 | **光影** | 整体**漫射光**（无强阴影、无强高光）——toon 风格前提 | ❌ 丁达尔光、❌ 镜面反射、❌ 强明暗对比 |

---

## 3. 图中应该出现的元素（3D 浮层将"挂"在图上）

| 区域 | 图中**应出现** | 3D 浮层叠加位置 |
|------|---------------|----------------|
| **天空区**（上 2/3）| 远山轮廓（柔和） / 远树剪影（**可有可无**）| 萤火虫随机分布 + 小鸟飞过 |
| **地平线**（中 1/3）| 平静湖面 / 河面 / 远处树林剪影——给画面"深度" | 路牌柱立于"地面"，正对湖面 / 远景 |
| **地面区**（下 1/3）| 草地（暖绿）/ 几块石 / 木材堆 / **篝火痕迹（强烈推荐）**| 3D 篝火将浮在你图的篝火痕迹上 |
| **前景**（底部）| **小路从画面底部边缘延伸到地平线** | 3D 引导线 Path 将沿这条路叠浮 |

## 4. 图中**不要**出现的元素

| 不要 | 原因 |
|------|------|
| ❌ **人 / 角色 / 露营者** | 猫角色是 3D 后加，不在图里 |
| ❌ **汽车 / 帐篷 / 现代物品** | 破坏氛围 |
| ❌ **路牌 / 指示牌** | 这是 3D 元素 |
| ❌ **文字 / 水印 / 签名 / Logo** | 干扰 + 难去除 |
| ❌ **明显的人工结构**（围栏 / 建筑 / 电线杆）| 破坏自然感 |

## 5. 3D 元素标定的"视觉锚点"（任务 2.23）

为了让 3D 元素能精确"挂"在图上，请尽量让图中**自然地**包含这些视觉锚点：

| 视觉锚点 | 我会在哪放置 3D 元素 | 你的图最好 |
|---------|---------------------|-----------|
| **地面与路牌的"接触点"** | 路牌柱底部（z = 0）| 地面**正中略偏前**（不是正中央、不是最底部）有清晰可辨的"平地" |
| **篝火痕迹** | 篝火 3D 模型（z = 1.0）| 地面**一侧**（左 / 右中）有"有火堆痕迹"的位置——石头圈 / 焦黑地 / 木柴堆皆可 |
| **小路起点** | Path 起点（z = 0.5）| 画面**正下方边缘**有"小路径" |
| **小路终点** | Path 终点 | 沿着"小路"向地平线延伸——3D 引导线将沿这条路叠浮 |
| **天空 1/2 高度** | 萤火虫随机分布 | 此区域**不要太满**（萤火虫要飘在干净的夜空上）|

> 💡 **如果你的图缺失某个锚点**（比如没有"篝火痕迹"），请告知，我会在 3D 标定时选择"画面中最合适的位置"并向你确认。

## 6. 不需要的内容（避免浪费图像信息量）

| 不要 | 原因 |
|------|------|
| ❌ 强烈季节感（雪 / 樱花 / 落叶堆）| 多图轮播时风格不统一 |
| ❌ 强反光（水面镜面 / 玻璃）| toon 风格无镜面反射 |
| ❌ 大量细节（草丛数量、树叶笔触）| 3D 浮层会"压住"细节 |
| ❌ 8-bit / 像素风格 / 油画厚重笔触 | 与"治愈系手绘"冲突 |
| ❌ 多焦点 / 复杂透视 | 干扰 3D 透视统一 |

## 7. 版权 / 来源确认

请确认以下任一：

- [ ] 图是你**自己拍摄 / 自己绘制**的（无版权问题）
- [ ] 图是**公有领域**（如 NASA、Unsplash 公有领域标记）
- [ ] 图是**你已获商用授权**的素材（如 Artlist / Pond5 等）
- [ ] 图是你有**特定使用授权**（如个人学习、非商用）

> ⚠️ **如果你不确定授权状态，先告诉我图的来源**，我再判断是否可用。

## 9. 我收到图后的处理流程（透明）

1. **确认授权 / 来源**（H6）
2. **校验尺寸 / 格式**（H1-H2）—— 不足 4K 我会用 AI 放大到 3840×2160
3. **转 webp**（目标 ≤ 1.5MB / 张）
4. **视觉锚点核对**（§5）—— 如果某锚点缺失，告知你并建议补图或调整 3D 位置
5. **放入 `public/scenes/`**：
   - 1 张 → `campsite-bg.webp`
   - 多张 → `campsite-bg-1.webp` / `-2.webp` / ...
6. **视觉回归对照**（任务 2.25）—— 在 Chrome 中实拍对照参考图

## 11. 验收清单（提交图前自检）

提交图前请用这张清单自检：

- [ ] 尺寸 ≥ 3840×2160
- [ ] 方向横向 16:9
- [ ] 地平线在画面 1/3 位置
- [ ] 正中央有可放路牌柱的"干净区"
- [ ] 暖色黄昏 / 湖边草地 / 低饱和度
- [ ] 远山 / 远景可见，给画面深度
- [ ] 篝火痕迹可见（强烈推荐）
- [ ] 小路从底部延伸到地平线
- [ ] 没有人 / 车 / 帐篷 / 路牌 / 文字 / 水印
- [ ] 版权状态已确认
